## strlen函数

```c
#include <string.h>

int main() {
    char arr9[] = "hello";
    int len = strlen(arr);
    printf("%d\n", len);
    return 0;
}
```

> 统计'\0'前面出现的字符个数

- 如果字符串里没有'\0',就一直往后面找,什么时候找到\0什么时候停止并输出

> 模拟实现

```c
#include <stdio.h>
#include <assert.h>

size_t my_strlen(const char *str) {
    size_t count = 0;
    assert(str);
    while (*str != '\0') {
        count++;
        str++;
    }
    return count;
}

int main() {
    char arr[] = "hello";
    size_t n = my_strlen(arr);
    printf("%zu\n", n);
    return 0;
}
```

## strcpy函数

```c
#include <stdio.h>
#include <string.h>

int main() {
    char name[20] = {0};
    strcpy(name, "zhangsan");
    printf("%s\n", name);
    return 0;
}
```

> strcpy 函数会将字符串里\0之前的内容复制到目标字符串中(包括\0)

- 源字符串里必须有\0,否则会出错
- 目标字符串必须有足够的空间,以确保能存放源字符串的内容
- 目标字符串必须可修改

```c
char *p = "hello";
char arr[] = "fewx";
strcpy(p, arr);
```

> 模拟实现

```c
#include <stdio.h>

char *my_strcpy(char *arr1, char *arr2, int sz) {
    for (int i = 0; i < sz; i++) {
        arr1[i] = arr2[i];
    }
    char *p = arr1;
    return p;
}

int main() {
    char arr1[] = "abc";
    char arr2[20] = {0};
    int sz = sizeof(arr1) / sizeof(arr1[0]);
    my_strcpy(arr2, arr1, sz);
    printf("%s\n", arr2);
    return 0;
}
```
