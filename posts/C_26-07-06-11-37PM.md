## 一维数组传参

```c
#include <stdio.h>

void test(int arr[]) {}
void test(int arr[10]) {}

void test(int *arr) {}
void test2(int *arr[20]) {}
void test2(int **arr) {}

int main() {
    int arr[10] = {0};
    int *arr2[20] = {0};
    test(arr);
    test2(arr2);
}
```
